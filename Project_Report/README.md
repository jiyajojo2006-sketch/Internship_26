# Project Report

## Project Overview
Sensor Linearization is a digital ASIC design project developed using Verilog HDL. The objective is to convert the non-linear output of a sensor into a linear output using piecewise linear approximation. The design was verified through simulation and implemented using the complete Cadence ASIC design flow.

## Objectives
- Design a sensor linearization circuit using Verilog HDL.
- Verify functionality through simulation.
- Perform logic synthesis.
- Complete the ASIC physical design flow.
- Analyze timing, area, and power.

## Design Methodology
1. RTL design using Verilog HDL.
2. Functional simulation using Cadence Xcelium.
3. Logic synthesis using Cadence Genus.
4. Floorplanning.
5. Power Planning.
6. Placement.
7. Clock Tree Synthesis (CTS).
8. Routing.
9. Signoff verification.
10. GDS generation.

## Results
- Functional simulation completed successfully.
- RTL synthesized successfully.
- Physical design flow completed.
- Timing, area, and power reports generated.
- Final layout prepared.
