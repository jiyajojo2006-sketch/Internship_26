# Sensor Linearization

## Project Overview
This project implements a digital sensor linearization circuit using Verilog HDL. The design converts non-linear sensor outputs into approximately linear outputs using piecewise linear approximation.

## Tools Used
- Verilog HDL
- Cadence Xcelium
- Cadence Genus
- Cadence Innovus

## Design Flow
- RTL Design
- Functional Simulation
- Synthesis
- Floorplanning
- Power Planning
- Placement
- Clock Tree Synthesis (CTS)
- Routing
- Signoff
- GDS Generation
